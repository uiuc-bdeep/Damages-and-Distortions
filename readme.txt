THE DAMAGES AND DISTORTIONS FROM DISCRIMINATION IN THE RENTAL HOUSING MARKET 
============================================================================

Code to carry out structural and reduced form analyses are written in Fortran and STATA, respectively.

Reduced form code, data and outputs of the reduced form anaylses can be found in "reduced form" folder.
Structural code, data and outputs can be found in "structural" folder.

Each folder has its own readme file with more details about the files contained in each folder.

The authors will assist with any reasonable replication attempts for two years following publication.
